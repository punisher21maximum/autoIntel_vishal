4. POC of an End-to-End IIoT Solution for Utility Industry

Description

Proof of Concept Development of an End-to-End ( Meter to Enterprise) IIoT Solution for an Utility backend Office. Two use case Implemented-
1.Read the meter Value
2.Trip the meter from the cloud if the prepaid amount exhausted

Client

Utility back-end Services Company, Pune

Salient Features

Retrofitting the existing Energy meter and converting into an Cost Effective Smart IIoT Energy Meter
Establishing P2P LoRa communication between Meter and Gateway and GPRS between Gateway to Cloud
Identification and customisation of a Gateway to suit the need
Read all the important 3 phase and 1 phase meter values ( V , C, Vrms, Active & Reactive Power, PF, Frequency, Totalised Power etc..) along with time stamp
Push the meter read request from the Cloud to the field
Seamless addition of devices in the field
Trip the prepaid meter from cloud if amount exhausted.
Tamper proof solution


Technology Involved

ESP32 with LoRa, Gateway based on Nordic Controller with GPRS interface, AWS IOT , MODBUS RTU, MQTT, Python
Embedded C/C++, AWS IOT

Responsibilities

Hardware Engineering, Solution Architecture, Design & Development of Embedded Application and Cloud Application
